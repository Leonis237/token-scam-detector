    How accurate is the scam detection? ▼
    Our neural network was trained on 200+ labeled tokens with ~81% cross-validation F1 score. It catches honeypots, rug pulls, and high-tax tokens. Always DYOR — this is a tool, not financial advice.
    Is this free to use? ▼
    Yes, completely free. We query the free Honeypot.is API and run inference on our own GPU. No wallet connection, no payment required.

    Nothing. We don't store addresses, IPs, or any personal data. The scan is stateless — your address never hits a database.

    Not yet. Honeypot.is currently supports EVM chains only. Solana support depends on their API expansion.
